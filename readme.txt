Deal per Order for HubSpot allows you to  automate the process of converting each potential sale into HubSpot deals. Grab a free automated system for your WooCommerce store to easily forecast your sales and revenue.

== Description ==

*Looking for an automated free system to convert every WooCommerce store order into HubSpot Deal and easily forecast your sales and revenue?*


Make a smart move by choosing  Deal Per Order for HubSpot. This plugin motivates the conversion of each order into the HubSpot deal automatically.

There are no hidden costs involved with the plugin installation for HubSpot CRM or any other helping extension. Also, extended support to HubSpot free CRM makes tracking much easier.

 Deal Per Order for HubSpot plugin connects your store to your HubSpot account and creates a special “WooCommerce Sales Order Pipeline”.  The order pipeline holds 6 stages and passes on every order to your HubSpot account in the Sales deal section. Every order is then converted into the HubSpot deal and is assigned to a contact and a valid deal stage.

**Note: Deal Per Order for HubSpot works even with HubSpot free CRM. It means you can get the full sales reporting feature in the free CRM as well.**

== WooCommerce Order Pipeline with Deal Stage Names ==

* Failed
* Canceled
* Pending
* On-hold
* Processed
* Shipped (completed orders)

== Main features ==

* Converts every order of your eCommerce store to HubSpot Deal with appropriate deal-stage, amount and closing date.

* Auto updates the deals on HubSpot whenever any order status is changed.

* Assigns the well-defined stage to the deals according to their current status on the store as order.

* **Deal Per Order for HubSpot** setup and creates a new “WooCommerce Order Pipeline” with 6 stages similar to woocommerce order status.

* **WooCommerce Sales Pipeline** uses different deal stages for the orders with the winning probability assigned to all.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of  Deal Per Order for HubSpot, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type " Deal Per Order for HubSpot" and click Search Plugins. Once you've found our  Deal Per Order for HubSpot plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking "Install Now".

= Manual installation =

The manual installation method involves downloading our  Deal Per Order for HubSpot and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.

== Frequently Asked Questions ==

= Will I get the HubSpot deals reporting and sales forecast with free CRM? =

Yes, HubSpot allows these two features even in the free CRM. You can get the deals to forecast from the sales dashboard section and can generate frequent reports based on that.

= How can I see all my new deals created by the extension? =

In the HubSpot Deals section, there is a filter for the pipelines and on selecting the WooCommerce Order Pipeline, it will show up all the new deals created under WooCommerce Order Pipeline.


= Is this plugin an add-on or works independently? =

The plugin  Deal Per Order for HubSpot is a single and main extension that works independently. All you have to do is to connect your HubSpot account and set up a new pipeline and you are ready with the plugin setup.


= How does it work? =

The plugin creates a new deal on HubSpot every time an order is placed on your store. Also on order status transition, the same will be reflected on HubSpot.
